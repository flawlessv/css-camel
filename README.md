### install

`npm install css-camel`

### usage

#### CSSToCamel

```
import {CSSToCamel} from 'css-camel'
CSSToCamel('background-color:pink') // 'backgroundColor:pink'

```

#### CamelToCSS

```
import {CamelToCSS} from 'css-camel'
CamelToCSS('backgroundColor:pink') // 'background-color:pink'

```
